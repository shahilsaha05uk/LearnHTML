In other words, it has some inherent meaning, the names have some inherent meaning. Now when it applies to HTML, what does semantic html element mean? Well, a semantic html element is an element that implies some meaning to the content. In other words, it's an element that tells you something about the content, whether its importance, whether it's a little bit of its description, it basically hints to you to that meaning. Now the reason that's important is because us, humans, and machines can understand the meaning of that content surrounded by a semantic tag much better than if we just surround it with some tag that doesn't really tell us anything other than this is some sort of a division within the document.
